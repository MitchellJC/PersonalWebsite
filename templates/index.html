  • Sitting Desktop Garden Prototype of a posture-tracking desk companion. Applies computer vision for face detection login and posture tracking. Plant "grows" (rises) when posture is good for a period of time. Plant "wilts" (descends) when posture is bad for a period of time. A connected mousepad gently vibrates when posture is first recognised as bad. Digital screen shows plot of historical posture data and assists with other user interaction such as logging in. Controlled by a Raspberry Pi and implemented using Python, SQLite, MediaPipe Pose, Dlib and PiicoDev.

    Personal contributions include a module for connecting to SQLite database to store posture data, asynchronous system for posture tracking with Google's MediaPipe API and applied face detection for login via saved embeddings using Dlib's Histogram of Oriented Gradients (HOG) model.
